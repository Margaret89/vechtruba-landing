Готовая сборка проекта лежит в папке dist. Просмотреть ее можно выложив на сервер. Локально не будут отображаться иконки и скрипты

Для запуска самого проекта необходимо в терминале прописать npm i для установке всех пакетов node modules. При этом на компьютере должен стоять Node.js не ниже 20 версии

Для запуска проекта в режиме правки пропишите команду npm run dev
Для сборки проекта пропишите команду npm run build
